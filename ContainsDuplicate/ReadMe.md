# Contains Duplicate #

---

### Brute force ###

> Time Comp O(n^2)

> Space Comp O(1)

the classic brute force approach, using 2 loops and checking the whole array one by one

---

### Optimized ###

> Time Comp O(n)

> Space Comp O(n)

The optimized sol uses a hashmap to reduce the Time Comp, as we check the map before adding a value
this sacrifices Space for saving time
